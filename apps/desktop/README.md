# 수소법률 RAG 데스크톱 앱

Tauri + React + Vite + Tailwind CSS

## 실행

```bash
npm install

# 개발 모드
npm run tauri:dev

# 빌드
npm run tauri:build
```

## 특징

- 가볍고 빠른 네이티브 앱
- 웹 앱 코드 재사용 (거의 100%)
- macOS, Windows, Linux 지원

## 번들 크기

- ~3MB (Electron 대비 40배 작음)
- 메모리 사용: ~50MB

## 백엔드 연동

백엔드 RAG 엔진: http://localhost:8000
